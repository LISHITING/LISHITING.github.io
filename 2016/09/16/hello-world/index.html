如何用Hexo搭建个人博客

发表于 2016-09-16 | 分类于 其他

如何配置环境

首先，根据你的系统是Mac还是Win会有些许不同，Mac需要安装Xcode以及Xcode里的命令后工具，因为hexo会需要编译工具，如果没有它的话就会报错。其他注意事项可以参考Hexo官网。确认系统类别以后就可以参考下面步骤啦：

  • 安装Git： 打开Git官网下载安装包安装。Git的存在是为了从Github上下载(或者叫git clone)文件目录 + 上传文章。当然还有很多强大的版本管理功能，但目前只关注博客要用的。git clone的方式下载会在文件夹里藏了一个.git文件用来保存文件版本信息。
  • 安装Node.js： 打开NodeJS官网下载安装包安装。Node.js为了安装和使用博客引擎。
  • 用npm安装hexo-cli：npm install -g hexo-cli。hexo-cli是hexo的命令后工具，通过在它里面输入各种命令就可以创建博客，创建博文，预览博客等等

至此，配置环境完成。

如何安装主题

安装主题一般步骤：

  • 在网站上找到一个主题。可以从hexo的主题推荐网站上找。下面以NextT为例
  • 找到该主题的github，然后按照提供的安装步骤进行。
  • 通常是git clone该主题，然后放在博客文件夹的themes子文件夹里。比如: $ git clone https://github.com/iissnan/hexo-theme-next themes/next
  • 打开博客的_config.yml文件并找到theme字段，修改对应的值。比如修改成next
  • 重启hexo服务器就可以看到新主题的样式

NextT主题的常用配置

添加文章类别

  • 首先需要用hexo建立一个页面:hexo new page categories
  • 然后打开source/categories/index.md
  • 添加一行type: "categories"， 注意是复数要有s
  • 通常不会让人在这个页面评论，所以再添加一行comments: false来禁止评论
  • 最后打开主题的_config.yml并将 menu 中的 categories: /categories 注释去掉。刷新页面就可以看到文章的类别了。

建议查看官方文档

如何添加文章标签

  • 基本同上，也是需要新建一个页面为tags
  • 然后打开source/tags/index.md
  • 添加一行type: "tags"， 注意是复数要有s
  • 添加一行comments: false来禁止评论
  • 打开主题的_config.yml并将 menu 中的 categories: /categories 注释去掉。刷新页面就可以看到文章的类别了。

如何插入数学公式

  • 打开主题的_config.yml
  • 定位到mathjax，把enable改成true

如何折叠首页内容

  • 打开主题的_config.yml
  • 定位到auto_excerpt，把enable改成true
  • 还可以修改预览字数，默认是150个字。
