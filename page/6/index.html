Introduction

This project is to make a stopwatch

Set up

Make sure you have nodejs and watchman installed

Make sure you have react-native cli installed.

Use react-native cli to Create a new project named stopwatch

Make sure you are in the project root folder, run npm start to start a server allow us to see the result.

Open Xcode project stopwatch/ios/stopwatch.xcodeproj, wait a little bit for loading the source and then hit run button.

Another way to start the project is go to the root folder and just run

Then it will start a server and a simulator automatically.

If you can see “Welcome to React Native!” and some text below, it means all the environment are set up.
