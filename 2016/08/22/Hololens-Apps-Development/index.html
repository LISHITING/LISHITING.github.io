Hololens Apps Development

2016-08-22

介绍

Microsoft Hololens 是微软开发增强现实的一款全息眼镜。其特点是既能够看真实世界又能够叠加虚拟信息的设备，并且也可以提供一个mic和小喇叭实现声音的交互。目前鉴于设备限制，采用最为便捷的Hololens Emulator 模拟器进行开发和测试。为了终端呈现用户数据和实现更为有效逼真的交互，目前开始着手基于Hololens的虚拟现实开发。以下是开发环境的建立过程。为确保能够正确开发和发布应用，请核实开发工具和开发环境的正确性。

开发工具

Visual Studio 2015 Built 3 下载
Microsoft Hololens Emulator 模拟器 下载
Unity 3D （5.4版本含VR的内容，关键） 下载

开发环境

Windows10 企业版物理机 （注：Windows10家庭版不支持）
64位，2核心扩展为4线程的CPU （4核CPU更佳）
8GB RAM内存
虚拟化开启：具体是 开启Hardware-assisted virtualization、Second Level Address Translation (SLAT) 和Hardware-based Data Execution Prevention (DEP)。
注：默认情况下如果Hyper-v正常运行，以上基本无需设置。
GPU：DirectX 11.0 或更高版本。 (此项可以没有，但经测试会影响虚拟器渲染的速度。)

可能存在的问题

关于服务器Microsoft windows server 2012R2上搭建会存在通用身份验证不被允许的问题。目前该问题无法解决。具体如下：
DEP6957 : 未能使用“通用身份验证”连接到设备“169.254.199.199”。请验证在项目调试设置中指定了正确的远程身份验证模式。0x8007274C: The network event being waited on triggered an error.
此消息将会在Visual Studio 发布至Hololens Emulator 虚拟起时报错。

参考
